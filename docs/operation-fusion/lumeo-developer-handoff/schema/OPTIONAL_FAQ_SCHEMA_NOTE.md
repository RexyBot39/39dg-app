# Optional FAQPage Schema Note

Visible FAQ sections are included in each HTML page. FAQPage JSON-LD is not embedded by default because FAQ rich-result eligibility and value may be limited for commercial pages. If SEO/legal chooses to deploy FAQPage schema, only mark up FAQ content that is visible to users on the page and avoid unsupported claims.

Developer: validate any FAQPage JSON-LD with Google's Rich Results Test and Schema Markup Validator before deployment.
